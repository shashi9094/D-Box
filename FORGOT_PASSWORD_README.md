## 🎯 User Flow (How It Works)

```
1. Login page → Click "Forgot Password?" link (aligned right below password field)
   ↓
2. forgot-password.html → Enter email → Click "Send Reset Code"
   ↓
3. Backend sends OTP via Brevo SMTP
   ↓
4. verify-otp.html → Enter 6-digit code from email
   ↓
5. Backend verifies OTP, generates reset token
   ↓
6. reset-password.html → Enter new password (with strength requirements)
   ↓
7. Backend hashes & updates password in database
   ↓
8. Redirect to login.html (success)
```

## 📋 API Endpoints

### POST /api/password-reset/forgot-password
```bash
curl -X POST http://localhost:5000/api/password-reset/forgot-password \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com"}'
```
**Response:** `{ "success": true, "messageId": "msg_id" }`

### POST /api/password-reset/verify-reset-otp
```bash
curl -X POST http://localhost:5000/api/password-reset/verify-reset-otp \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com","otp":"123456"}'
```
**Response:** `{ "success": true, "token": "reset_token_hash" }`

### POST /api/password-reset/reset-password
```bash
curl -X POST http://localhost:5000/api/password-reset/reset-password \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com","token":"token_hash","password":"NewPass123"}'
```
**Response:** `{ "success": true, "message": "Password reset successfully" }`
